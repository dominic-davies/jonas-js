Lecture 2.8 - Variables and data types

Primitive data types (non object)

  1. Number: in js, always floating point, always have decimals even if you don't see them (or look like integers).
  2. String: text, a sequence of characters
  3. Boolean: Logical data type, can only be true or false.
  4. Undefined: gets automatically assigned to a var that has no value yet
  5. Null: 'non existent'

JS has dynamic typing.

Data types are automatically assigned to variables. you don't have to say what type it is js knows that "john" is a string and 28 is a number.

Variable names

Give variables meaningful names. Use camelCase notation. Can't start a var with a number.
